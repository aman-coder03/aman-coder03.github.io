# Personal Website (Personal Portfolio)

This repository contains the source code for a **personal portfolio website** hosted using **GitHub Pages (`github.io`)**.

The website is built using **Markdown with HTML styling** and is compatible with **Jekyll**, making it easy to customize and deploy.

## Template Usage

This repository includes a reusable template:
`template.md`


You can **copy this template and use it to build your own personal website**.

### How to use the template

1. Copy `template.md`
2. Rename it to `index.md`
3. Replace the placeholder values with your own content
4. Push the changes to your GitHub repository

Once pushed, GitHub Pages will automatically build and host your site.

---

## Hosting on GitHub Pages

To host your site using GitHub Pages:

1. Fork or clone this repository
2. Go to **Settings â†’ Pages**
3. Select the `main` branch as the source
4. Set the root directory (`/`) as the publish location
5. Save, your site will be live at `https://<username>.github.io`

---

## Customization

- Edit `index.md` to update content
- Modify styles directly in Markdown/HTML
- Add new pages as additional `.md` files
- Use `template.md` as a starting point for new pages
